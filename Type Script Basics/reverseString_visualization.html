🔄 reverseString() Function

Visual Guide to String Reversal Logic

📋 Function Overview

function reverseString(str: string): string { const reversedStr = str.split('').reverse().join(''); const result = `The reversed string of "${str}" is: "${reversedStr}"`; console.log(result); return result; }

This function takes a string as input, reverses it, formats a message, logs it to the console, and returns the formatted result.

⚙️ Logic Flow

1
Input: Receive string parameter (e.g., "hello")
↓
2
Split: Convert string to array using .split('')
↓
3
Reverse: Reverse array order using .reverse()
↓
4
Join: Convert array back to string using .join('')
↓
5
Format: Create formatted message with original and reversed strings
↓
6
Output: Log to console and return the result

🔍 Step-by-Step Breakdown

1
split('') - Convert String to Array
Splits the string into an array of individual characters
Input: "hello"
Result: ['h', 'e', 'l', 'l', 'o']
2
reverse() - Reverse Array Order
Reverses the order of elements in the array
Input: ['h', 'e', 'l', 'l', 'o']
Result: ['o', 'l', 'l', 'e', 'h']
3
join('') - Convert Array Back to String
Joins all array elements into a single string without separator
Input: ['o', 'l', 'l', 'e', 'h']
Result: "olleh"
4
Format Message
Create a formatted string with the original and reversed strings
Template: `The reversed string of "${str}" is: "${reversedStr}"`
Result: The reversed string of "hello" is: "olleh"
5
Console Log & Return
Print the formatted message to console and return the result
Output: The reversed string of "hello" is: "olleh"

🔑 Key Methods Used

split(separator)

Splits a string into an array of substrings. Using '' splits into individual characters.

reverse()

Reverses an array in-place. The first element becomes the last, and vice versa.

join(separator)

Joins all array elements into a string. Using '' concatenates without separators.
